```java
    .orderBy(gnb.gnbDetails.displayOrder.asc())
    .transform(
        GroupBy.groupBy(gnb.id)
            .list(new QGnbResponse(...)))
```

### WHERE 조건

| 조건 | 설명 |
|------|------|
| betweenTime | 현재 시간이 전시 기간 내 |
| deleteYnNo | gnb.deleteYn = 'N' |
