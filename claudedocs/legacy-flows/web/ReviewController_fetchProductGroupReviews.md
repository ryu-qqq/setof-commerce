queryFactory
    .from(review)
    .innerJoin(orderSnapShotProductGroup)
        .on(orderSnapShotProductGroup.orderId.eq(review.orderId))
    .leftJoin(productScore)
        .on(productScore.id.eq(orderSnapShotProductGroup.snapShotProductGroup.productGroupId))

## 📊 성능 고려사항

1. **2단계 쿼리 패턴**: ID 조회 → 상세 조회 (N+1 방지)
2. **GroupBy Transform**: 1:N 관계 데이터를 메모리에서 그룹핑
3. **평균 평점 별도 테이블**: `product_rating_stats`에 사전 집계
4. **정렬 옵션**: productScore 조인으로 추천순 정렬 지원
