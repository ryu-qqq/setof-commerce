### FULLTEXT 검색 조건

```java
// MySQL ngram FULLTEXT 인덱스 사용
// 인덱스: ALTER TABLE product_group ADD FULLTEXT INDEX ft_product_group_name (product_group_name) WITH PARSER ngram;

private BooleanExpression fullTextSearch(String searchWord) {
    if (searchWord == null || searchWord.isBlank()) {
        return null;
    }
    // Hibernate 6 커스텀 함수: match_against
    return Expressions.numberTemplate(
            Double.class,
            "function('match_against', {0}, {1})",
            productGroup.productGroupDetails.productGroupName,
            "+" + searchWord + "*"
    ).gt(0);
}
```

### 커서 페이징 조건 (OrderType별)

| OrderType | 커서 조건 |
|-----------|----------|
| RECOMMEND | score < cursorValue OR (score = cursorValue AND id <= lastDomainId) |
| REVIEW | reviewCount < cursorValue OR (reviewCount = cursorValue AND id <= lastDomainId) |
| RECENT | insertDate < cursorValue OR (insertDate = cursorValue AND id <= lastDomainId) |
| HIGH_RATING | averageRating < cursorValue OR (averageRating = cursorValue AND id <= lastDomainId) |
| HIGH_PRICE | salePrice < cursorValue OR (salePrice = cursorValue AND id <= lastDomainId) |
| LOW_PRICE | salePrice > cursorValue OR (salePrice = cursorValue AND id >= lastDomainId) |
| HIGH_DISCOUNT | discountRate < cursorValue OR (discountRate = cursorValue AND id <= lastDomainId) |
| LOW_DISCOUNT | discountRate > cursorValue OR (discountRate = cursorValue AND id >= lastDomainId) |
