// 브랜드명 검색 - mainDisplayType에 따라 검색 필드 결정
private BooleanExpression isBrandNameLike(BrandFilter filter) {
    if (StringUtils.hasText(filter.getBrandName())) {
        if (filter.getMainDisplayType().isKorBrandName()) {
            return brand.displayKoreanName.contains(filter.getBrandName());
        } else {
            return brand.displayEnglishName.contains(filter.getBrandName());
        }
    }
    return null;
}

// No-Offset 조건 - 정렬 방향에 따라 gt/lt 결정
private BooleanExpression getBrandIdCondition(BrandFilter filter, Pageable pageable) {
    Sort.Order order = pageable.getSort().getOrderFor("id");
    if (order != null && order.isAscending()) {
        return isBrandIdGt(filter);  // brand.id > lastDomainId
    } else {
        return isBrandIdLt(filter);  // brand.id < lastDomainId
    }
}

## 🔍 특이사항

### 1. No-Offset 페이징 지원
- `lastDomainId` 파라미터로 커서 기반 페이징 지원
- 대용량 데이터에서 성능 최적화 가능

### 2. 다국어 검색
- `mainDisplayType`에 따라 한글(`displayKoreanName`) 또는 영문(`displayEnglishName`) 필드 검색
- 동적 필드 선택 로직

### 3. 미사용 필드
- `siteId`, `brandNames` 필드는 Filter에 존재하나 쿼리에서 미사용
