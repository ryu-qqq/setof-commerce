### QueryDSL 코드

```java
queryFactory
    .from(bannerItem)
    .join(banner).on(banner.id.eq(bannerItem.bannerId))
    .where(
        bannerIdEq(bannerId),
        betweenTime(filterDto),
        bannerItem.deleteYn.eq(Yn.N)
    )
    .orderBy(bannerItem.displayOrder.asc())
    .transform(
        GroupBy.groupBy(bannerItem.id).list(
            new QBannerItemDto(
                bannerItem.id,
                banner.bannerType,
                bannerItem.title,
                bannerItem.displayPeriod,
                bannerItem.imageUrl,
                bannerItem.linkUrl,
                bannerItem.displayOrder,
                bannerItem.displayYn,
                bannerItem.imageSize
            )
        )
    );
```
