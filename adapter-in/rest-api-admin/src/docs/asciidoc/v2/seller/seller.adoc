=== Seller (셀러 관리)

셀러 등록, 조회, 수정, 승인/거절/정지/삭제 기능을 제공합니다.

==== 셀러 목록 조회

`GET /api/v2/admin/sellers`

검색 조건에 맞는 셀러 목록을 페이징하여 조회합니다.

operation::admin-seller-list[snippets='http-request,query-parameters,http-response,response-fields']

==== 셀러 상세 조회

`GET /api/v2/admin/sellers/{sellerId}`

셀러 ID로 상세 정보를 조회합니다.

operation::admin-seller-detail[snippets='http-request,path-parameters,http-response,response-fields']

==== 셀러 등록

`POST /api/v2/admin/sellers`

새로운 셀러를 등록합니다. 초기 상태는 PENDING입니다.

operation::admin-seller-register[snippets='http-request,request-fields,http-response,response-fields']

==== 셀러 수정

`PUT /api/v2/admin/sellers/{sellerId}`

셀러 정보를 수정합니다.

operation::admin-seller-update[snippets='http-request,path-parameters,request-fields,http-response,response-fields']

==== 셀러 승인

`PATCH /api/v2/admin/sellers/{sellerId}/approve`

대기 상태의 셀러를 승인합니다.

operation::admin-seller-approve[snippets='http-request,path-parameters,http-response,response-fields']

==== 셀러 거절

operation::admin-seller-reject[snippets='http-request,path-parameters,http-response,response-fields']

==== 셀러 정지

`PATCH /api/v2/admin/sellers/{sellerId}/suspend`

승인된 셀러를 정지합니다.

operation::admin-seller-suspend[snippets='http-request,path-parameters,http-response,response-fields']

==== 셀러 삭제
