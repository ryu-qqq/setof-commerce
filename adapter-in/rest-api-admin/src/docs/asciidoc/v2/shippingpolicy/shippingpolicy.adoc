=== Shipping Policy (배송 정책 관리)

셀러별 배송 정책 등록, 조회, 수정, 기본 정책 설정, 삭제 기능을 제공합니다.

==== 배송 정책 목록 조회

`GET /api/v2/admin/sellers/{sellerId}/shipping-policies`

셀러의 배송 정책 목록을 조회합니다.

operation::admin-shipping-policy-list[snippets='http-request,path-parameters,http-response,response-fields']

==== 배송 정책 상세 조회

`GET /api/v2/admin/sellers/{sellerId}/shipping-policies/{shippingPolicyId}`

배송 정책 ID로 상세 정보를 조회합니다.

operation::admin-shipping-policy-detail[snippets='http-request,path-parameters,http-response,response-fields']

==== 배송 정책 등록

`POST /api/v2/admin/sellers/{sellerId}/shipping-policies`

새로운 배송 정책을 등록합니다.

operation::admin-shipping-policy-register[snippets='http-request,path-parameters,request-fields,http-response,response-fields']

==== 배송 정책 수정

`PUT /api/v2/admin/sellers/{sellerId}/shipping-policies/{shippingPolicyId}`

배송 정책 정보를 수정합니다.

operation::admin-shipping-policy-update[snippets='http-request,path-parameters,request-fields,http-response,response-fields']

==== 기본 배송 정책 설정

`PATCH /api/v2/admin/sellers/{sellerId}/shipping-policies/{shippingPolicyId}/default`

해당 배송 정책을 기본 정책으로 설정합니다.

operation::admin-shipping-policy-set-default[snippets='http-request,path-parameters,http-response,response-fields']

==== 배송 정책 삭제

`PATCH /api/v2/admin/sellers/{sellerId}/shipping-policies/{shippingPolicyId}/delete`

배송 정책을 삭제합니다 (소프트 삭제).

operation::admin-shipping-policy-delete[snippets='http-request,path-parameters,http-response,response-fields']
