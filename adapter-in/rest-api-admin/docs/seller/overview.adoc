[[seller-overview]]
= 셀러 관리 시스템 개요

셀러(Seller) 관리 시스템은 입점 신청부터 셀러 운영까지의 전체 라이프사이클을 관리합니다.

== 시스템 구성

셀러 관리 시스템은 크게 두 가지 도메인으로 구성됩니다:

[cols="1,2,2"]
|===
| 도메인 | 설명 | 주요 기능

| **SellerApplication**
| 입점 신청 관리
| 입점 신청, 승인/거절 처리

| **Seller**
| 셀러 정보 관리
| 셀러 정보 조회/수정, 정책 관리
|===

== 셀러 정보 구조

승인된 셀러는 다음과 같은 정보를 관리합니다:

=== 기본 정보 (Seller)

[cols="1,2"]
|===
| 항목 | 설명

| 셀러명 | 셀러 고유 이름 (중복 불가)
| 표시명 | 고객에게 노출되는 브랜드명
| 로고 URL | 셀러 로고 이미지
| 설명 | 셀러 소개 문구
| 활성화 여부 | 셀러 운영 상태
|===

=== 주소 정보 (SellerAddress)

[cols="1,2"]
|===
| 항목 | 설명

| 주소 타입 | RETURN(반품), EXCHANGE(교환), SHIPPING(출고)
| 주소명 | 주소 별칭 (예: 본사, 물류센터)
| 우편번호 | 5자리 우편번호
| 주소 | 기본 주소
| 상세주소 | 상세 주소
| 담당자명 | 해당 주소 담당자
| 담당자 연락처 | 담당자 전화번호
|===

=== 사업자 정보 (SellerBusinessInfo)

[cols="1,2"]
|===
| 항목 | 설명

| 사업자등록번호 | 10자리 사업자등록번호 (중복 불가)
| 회사명 | 법인명 또는 상호명
| 대표자명 | 사업자 대표자
| 통신판매업 신고번호 | 통신판매업 신고 번호
| 사업장 주소 | 사업자등록증 상 주소
|===

=== CS 정보 (SellerCs)

[cols="1,2"]
|===
| 항목 | 설명

| 고객센터 전화번호 | 대표 전화번호
| 고객센터 휴대폰번호 | 담당자 휴대폰
| 고객센터 이메일 | CS 이메일
| 운영 시간 | 고객센터 운영 시작/종료 시간
| 운영 요일 | 운영 요일 (예: MON,TUE,WED,THU,FRI)
| 카카오 채널 URL | 카카오톡 채널 URL
|===

=== 계약 정보 (SellerContract)

[cols="1,2"]
|===
| 항목 | 설명

| 수수료율 | 판매 수수료율 (%)
| 계약 시작일 | 계약 시작 날짜
| 계약 종료일 | 계약 종료 날짜
| 계약 상태 | PENDING, ACTIVE, EXPIRED, TERMINATED
| 특별 조건 | 특별 계약 조건
|===

=== 정산 정보 (SellerSettlement)

[cols="1,2"]
|===
| 항목 | 설명

| 은행 코드 | 은행 식별 코드
| 은행명 | 은행 이름
| 계좌번호 | 정산 계좌번호
| 예금주명 | 계좌 예금주
| 정산 주기 | WEEKLY, BIWEEKLY, MONTHLY
| 정산일 | 정산 실행일 (1-31)
| 인증 여부 | 계좌 인증 완료 여부
|===
