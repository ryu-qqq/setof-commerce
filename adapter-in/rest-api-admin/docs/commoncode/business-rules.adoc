= 공통 코드 비즈니스 규칙

이 문서는 공통 코드 시스템의 핵심 비즈니스 규칙을 정의합니다.

== 공통코드 타입 규칙

=== 등록 규칙

[cols="1,3,2"]
|===
| 규칙 ID | 규칙 | 설명

| CCT-REG-001
| 코드는 시스템 내에서 고유해야 합니다
| 중복 코드 등록 불가

| CCT-REG-002
| 코드는 영문 대문자와 언더스코어만 허용
| 예: `PAYMENT_METHOD`, `ORDER_STATUS`

| CCT-REG-003
| 코드는 등록 후 변경할 수 없습니다
| 다른 시스템에서 참조할 수 있으므로 불변
|===

=== 수정 규칙

[cols="1,3,2"]
|===
| 규칙 ID | 규칙 | 설명

| CCT-UPD-001
| 이름(name)은 변경 가능합니다
| 표시용이므로 변경 허용

| CCT-UPD-002
| 설명(description)은 변경 가능합니다
|

| CCT-UPD-003
| 표시 순서(displayOrder)는 변경 가능합니다
|
|===

=== 상태 변경 규칙

[cols="1,3,2"]
|===
| 규칙 ID | 규칙 | 에러 코드

| CCT-STS-001
| 하위 공통코드가 모두 비활성화되어야 타입 비활성화 가능
| `ACTIVE_CODES_EXIST`

| CCT-STS-002
| 비활성화된 타입의 공통코드는 조회에서 제외
|
|===

== 공통코드 규칙

=== 등록 규칙

[cols="1,3,2"]
|===
| 규칙 ID | 규칙 | 설명

| CC-REG-001
| 동일 타입 내에서 코드는 고유해야 합니다
| 다른 타입에서는 동일 코드 사용 가능

| CC-REG-002
| 코드는 영문 대문자와 언더스코어만 허용
| 예: `CREDIT_CARD`, `PENDING`

| CC-REG-003
| 코드는 등록 후 변경할 수 없습니다
| 다른 시스템에서 참조할 수 있으므로 불변

| CC-REG-004
| 활성화된 타입에만 공통코드 등록 가능
| 비활성 타입에는 등록 불가
|===

=== 수정 규칙

[cols="1,3,2"]
|===
| 규칙 ID | 규칙 | 설명

| CC-UPD-001
| 표시명(displayName)은 변경 가능합니다
| 사용자에게 보여지는 이름

| CC-UPD-002
| 표시 순서(displayOrder)는 변경 가능합니다
|
|===

=== 상태 변경 규칙

[cols="1,3,2"]
|===
| 규칙 ID | 규칙 | 설명

| CC-STS-001
| 공통코드는 언제든지 비활성화 가능합니다
| 기존 데이터에는 영향 없음

| CC-STS-002
| 비활성화된 코드는 신규 선택에서 제외됩니다
| 조회 시 active 필터 적용
|===

== 코드 형식 규칙

[cols="2,3,2"]
|===
| 구분 | 허용 형식 | 예시

| 영문 대문자
| A-Z
| `PAYMENT`, `ORDER`

| 언더스코어
| _
| `PAYMENT_METHOD`

| 시작 문자
| 영문 대문자로 시작
| `ORDER_STATUS` (O), `_ORDER` (X)

| 최소 길이
| 2자 이상
| `OK` (O), `A` (X)

| 최대 길이
| 50자 이하
|
|===

== 에러 코드 정의

=== 공통코드 타입 관련 에러

[cols="2,1,3"]
|===
| 에러 코드 | HTTP 상태 | 메시지

| `COMMON_CODE_TYPE_NOT_FOUND`
| 404
| 공통코드 타입을 찾을 수 없습니다

| `COMMON_CODE_TYPE_CODE_DUPLICATE`
| 400
| 이미 존재하는 공통코드 타입 코드입니다

| `COMMON_CODE_TYPE_INACTIVE`
| 400
| 비활성화된 공통코드 타입입니다

| `ACTIVE_CODES_EXIST`
| 400
| 활성화된 공통코드가 존재하여 타입을 비활성화할 수 없습니다
|===

=== 공통코드 관련 에러

[cols="2,1,3"]
|===
| 에러 코드 | HTTP 상태 | 메시지

| `COMMON_CODE_NOT_FOUND`
| 404
| 공통코드를 찾을 수 없습니다

| `COMMON_CODE_DUPLICATE`
| 400
| 동일 타입 내에 이미 존재하는 코드입니다

| `INVALID_CODE_FORMAT`
| 400
| 코드 형식이 올바르지 않습니다 (영문 대문자, 언더스코어만 허용)
|===

== 조회 필터링 규칙

=== 기본 조회 동작

[cols="2,4"]
|===
| 필터 | 설명

| `active` 미지정
| 모든 상태 조회 (활성 + 비활성)

| `active=true`
| 활성화된 항목만 조회

| `active=false`
| 비활성화된 항목만 조회
|===

=== 정렬 규칙

[cols="2,4"]
|===
| 정렬 키 | 설명

| `CREATED_AT`
| 생성일 기준 정렬 (기본값)

| `DISPLAY_ORDER`
| 표시 순서 기준 정렬 (UI 표시용)

| `CODE`
| 코드 알파벳 순 정렬
|===
